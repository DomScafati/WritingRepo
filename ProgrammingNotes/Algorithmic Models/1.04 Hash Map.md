**Time Complexity:** linear, O(n)
**Space Complexity:** constant, O(n)

The second most important algorithm to practice is a personal favorite. The Hash Map algorithm is a bit of a catch-all for any loop utilizing a key/pair value collection. In Swift, it is known as a *dictionary*, so going forward we will use this terminology.

One of the great aspects of using dictionaries is that they are very versatile and can be used in many different coding challenges, as it does not require the collection be sorted. The ability to perform a lookup makes this a great option when you are in need of counting elements or looking for duplicates.

**Ex.  Hash Map
**Goal**: Given an array, return the number that appears most often.

```
func mostFrequent(_ array: [Int]) -> Int? {
    if array.isEmpty {
        return nil
    }
    
    var dict: [Int: Int] = [:]
    var mostFrequentNumber = array[0]
    var highestCount = 0
    
    for num in array {
        dict[num, default: 0] += 1
        let currentCount = dict[num]!
        
        if currentCount > highestCount {
            highestCount = currentCount
            mostFrequentNumber = num
        }
    }
    
    return mostFrequentNumber
}
```

**Breakdown**:
1. If the array is empty, return `nil`.
2. Create a dictionary to store counts.
3. Track two variables:
    - `highestCount` (the largest frequency seen so far)
    - `mostFrequentNumber` (the value associated with that frequency)
4. Iterate through the array once.
5. Increment the count for the current number.
6. If its updated count exceeds `highestCount`, update both trackers.
7. Return the most frequent value at the end.
