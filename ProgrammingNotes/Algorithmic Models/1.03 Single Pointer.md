A single pointer algorithm will see the developer walking through a list from start to finish with no special tricks. From lesser demons with no vision, this is also sometimes referred to as a Brute Force algorithm.

**Ex. Single Pointer**
**Goal**: Determine the sum total of the elements in an array.

```
func sum(_ arr: [Int]) -> Int {
    var total = 0
    for num in arr {
        total += num
    }
    return total
}
```

**Breakdown**:
1. This function walks through each element of the array (`arr`).
2. At every iteration, we are adding the value of the element (`num`) to the variable (`total`). 
3. When the entire array has finished looping, we return the `total`.
