**Time Complexity:** linear, O(n)  
**Space Complexity:** constant, O(1)

The two pointer algorithm is a powerful way to cleanly sort through an array. One pointer is at the beginning of a collection, the left side, whereas the the second pointer is at the end of a collection, the right side. 

**Heed these words:** This technique only works correctly when the collection is sorted. If the array is not sorted, seek other methods.

**Ex. Two Pointer**  
**Goal**: Given a sorted array, find two numbers whose sum equals a target value and return their indices.

```
func pairSum(_ nums: [Int], target: Int) -> [Int] {
    var left = 0
    var right = nums.count - 1
    
    while left < right {
        let sum = nums[left] + nums[right]
        
        if sum == target {
            return [left, right]
        } else if sum < target {
            left += 1
        } else {
            right -= 1
        }
    }
    
    return [-1, -1]
}
```

**Breakdown**:
1. We initialize two pointers: one at the beginning (`left`) and one at the end (`right`) of the array.
2. While the pointers have not crossed, we calculate the sum of the values at those positions.
3. If the sum equals the target, we return the indices immediately.
4. If the sum is too small, we move the `left` pointer forward to increase the sum.
5. If the sum is too large, we move the `right` pointer backward to decrease the sum.
6. If the pointers meet, no valid pair exists.
