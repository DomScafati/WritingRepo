**Time Complexity:** Logarithmic O(log n)
**Space Complexity:** Constant, O(1)

Ah yes, the Binary Search, the art of elimination. Many find this algorithm to be a favorite for it's simple, yet clever nature. Rather than scanning every element, we shall repeatedly cut the collection in half. Each iteration discards 50% of the possibilities until only one remains.

**Heed these words:** This is another technique that only works correctly when the collection is sorted. If the array is not sorted, seek other methods.

**Ex. Binary Search**
**Goal:** Find out if the target integer exists in the array. If it does, return true. If it does not, return false.

```
func findTargetBinarySearch(_ array: [Int], target: Int) -> Bool {
    var left = 0
    var right = array.count - 1
    
    while left <= right {
        var middle = (left + right) / 2
        let guess = array[middle]
        
        if guess == target {
            return true
        } else if guess > target {
            right = middle - 1
        } else if guess < target {
            left = middle + 1
        }
    }
    
    return false
}
```

**Breakdown:**
1. We initialize two boundaries: `left` at the start of the array and `right` at the end. These define the current searchable region.
2. While the search region is valid (`left <= right`), we compute the `middle` index. This splits the remaining elements in half.
3. We inspect the value at `middle` (`guess`).
4. If `guess` equals the target, the search is complete.
5. If `guess` is greater than the target, we discard the entire right half by moving `right` to `middle - 1`.
6. If `guess` is less than the target, we discard the entire left half by moving `left` to `middle + 1`.
7. If the boundaries cross, the element does not exist in the array.

**Q:** Drak' Tharr, the Pit Fiend, why do we need `+ 1` and `- 1` respectively?

In this specific example, removing the `+ 1` and `- 1` would change very little, however, when coding it's best to think of the highest and lowest extremes. What if there were 1 million elements in the collection, and we didn't do this? We would waste a monolithic amount of time and computing power.
